# Simple CLI Calculator

A simple command-line calculator built with Python that performs basic arithmetic operations safely and interactively.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles invalid input gracefully
- Prevents division by zero
- Supports decimal numbers
- Allows chained calculations
- Help command for usage instructions

## How It Works
The calculator runs in a loop and prompts the user for:
1. First number
2. Operator
3. Second number

The program continues running until the user chooses to quit.
